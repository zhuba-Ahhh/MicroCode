# MicroCode

## 欢迎使用 MicroCode

**MicroCode** 是基于[Mybricks](https://github.com/mybricks)企业级低代码设计引擎开发，仅供本人毕业设计使用。

## 开始使用

建议使用 `pnpm`

### 安装`pnpm`

```bash
npm i pnpm -g
```

### 安装依赖

```bash
pnpm install
```

### 本地调试

```bash
pnpm dev
```

[open 查看](http://localhost:8000/page-pc)

### 打包

```bash
pnpm build
```

### 查看效果

```bash
pnpm i serve -g 

serve dist -l 4005
```

[open 查看](http://localhost:4005/page-pc)
